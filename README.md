## Dataset

This project uses a public dataset from Google BigQuery:

Dataset: thelook_ecommerce  
Table: products

Source:
https://console.cloud.google.com/marketplace/product/bigquery-public-data/thelook-ecommerce

The dataset contains information about e-commerce products including:

- product name
- brand
- category
- retail price

The SQL queries in this project analyze:

- number of products by category
- average price by category
- most popular brands
- most expensive products
- price distribution across categories
